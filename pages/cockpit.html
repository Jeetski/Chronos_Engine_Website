Cockpit

The Cockpit view is a drag and drop canvas for modular panels. It is designed as a personal flight deck where you can dock the data you care about most.

Core Behaviors

  • Pan by dragging empty space, scroll to move, or use the floating controls.
  • Zoom with Ctrl + scroll or the zoom buttons.
  • Snap grid can be toggled from the toolbar.
  • Layout is stored in localStorage under chronos_cockpit_panels_v1.

Panels

  • Schedule - live agenda tree for Today.
  • Matrix - pivot style grid across items and properties.
  • Matrix Visuals - chart view of Matrix data.
  • Status Strip - current status indicators in a horizontal strip.
  • Commitments - snapshot of commitment status and triggers.
  • Lists - quick list slices of Chronos data.
  • Deadlines - upcoming due dates and schedule pressure.
  • Map of Happiness - mood and energy visualization.
  • Checklist - lightweight checklist panel for quick capture.
  • Flashcards - practice cards drawn from notes and tags.
  • Random Picker - spin a choice from a list.

How Panels Register

Panels live under Utilities/Dashboard/Panels/<Name> and register via register(manager). The Cockpit manager exposes a window.__cockpitPanelRegister hook to add panels without touching HTML.

Theme Support

Cockpit panels inherit shared theme variables from the dashboard so palettes stay consistent across views.
