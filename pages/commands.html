Command Reference

Status: Early development — this list reflects current commands detected in the repo. Developer: David Cody (Hivemind Studio).

Chronos CLI commands in concise form. Arguments in <angle brackets> are required; items in [square brackets] are optional.

add <item> to <target> [position:<number>]

Adds an item to a template's sub-items.

append <type> <name> "text"

Appends text to an item's content.

change <name> <HH:MM>

Changes an item's start time in today's schedule.

cls

Clears the terminal screen.

cmd <command>

Executes a command-line command.

complete

Completes items based on context (when enabled).

copy <type> <source> [new]

Duplicates an existing item.

count <type> [prop:value ...]

Counts items, optionally filtered.

create|new <type> <name> [prop:value ...]

Creates a new item.

cut <name>

Removes an item from today's schedule.

dashboard

Opens the Chronos web dashboard.

delete [-f|--force] <type> <name>

Deletes an item.

dismiss|skip|snooze <alarm_name>

Manages alarms for the current day.

echo "text"

Prints text to the console.

edit <type> <name> [editor:<name>]

Opens an item in an editor.

export

Exports data. Use export all [filename.zip] to zip the entire User/ folder into User/Exports/. Or export <file.yml> list <type> [props...] to save a table as YAML.

filter <type> [prop:value ...] | filter all | filter off

Sets or clears the active item filter.

find <type> <keyword> [prop:value ...]

Searches for items.

get <type> <name> <property>

Retrieves a specific property value.

help [command]

Shows help for the CLI or a specific command.

import

Imports data. If the path ends with .zip, extracts a full backup into User/ (overwrite:true to replace). If it's a YAML list, creates items (skips existing).

list <type> [sort_by:<key>] [reverse_sort:True|False]

Lists items with sorting options.

mark <name>:<status>

Marks an item in the daily schedule with a status.

mark_norm

Normalizes marks/status for items where supported.

miss

Manages or lists missed alarms/slots (when available).

move <src_type> <src_name> [new] [type:<dst_type>]

Moves and/or renames an item.

pause [message]

Pauses script execution.

powershell <code>

Runs PowerShell code.

remove <type> <name> <property>

Removes a property from an item.

rename <type> <old> <new>

Renames an item.

if <left> <op> <right> then <command> [else <command>]

Conditional execution. Operators: = != > < >= <= eq ne gt lt ge le matches; Logic: and or xor nor not !; Supports exists for items, file:, dir:, env:.

set <type> <name> <key>:<value> [...]

Sets one or more properties.

vars [name:<var>]

Lists current script variables (see set var). Use @name to interpolate in commands.

unset var <name>

Removes a script variable from the current session.

settings <file_shortcut> <property> <value>

Modifies a setting.

status [type:value]

Views or sets status variables.

today [reschedule]

Displays or regenerates today's schedule.

theme list|current|set <name> | set-colors background:<name|#hex> text:<name|#hex>

Manages console theme. Themes live in User/Settings/theme_settings.yml; profile overrides in User/profile.yml. On Windows, applies via color.

commitments [check]

Evaluates commitment rules and fires triggers. Supports frequency goals (times per day/week/month) and never rules with forbidden_items. Triggers can create achievement/reward items or run scripts.

track

Starts/stops tracking utilities (where available).

trim <name> <minutes>

Reduces an item's duration today.

view <type> <name>

Displays an item's content and properties.
