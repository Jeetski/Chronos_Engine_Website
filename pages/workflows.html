Common Workflows

Practical, repeatable ways to use Chronos â€” for humans and agents.

Capture & Organize

new task "Buy milk" priority:medium
append note "Idea: Morning pages" "Try 3 pages after coffee."
new project "Home Gym"
new task "Order dumbbells" project:"Home Gym"
  • Defaults: add task_defaults.yml to set default_priority/default_status.
  • View/edit: view task "Buy milk", edit task "Buy milk".

Templates & Routines

new routine "Morning"
add "Have coffee" to "Morning"
add "Read 10 min" to "Morning"
  • Nesting: use subroutines and microroutines for structure.
  • Clone: copy routine "Morning" "Morning (travel)".

Plan the Day

status energy:high focus:good
today reschedule
trim "Read 10 min" 5
change "Have coffee" 07:45
  • Mark: mark "Have coffee":completed.
  • Dependencies: add depends_on in item YAML to hint order.

Alarms & Reminders

new reminder "Stand up" time:15:00 recurrence:[daily] enabled:true
  • Target actions: set target: { type, name, action, properties } in the YAML.
  • Snooze/dismiss: snooze "Stand up", dismiss "Stand up".

Goals & Habits

new goal "Publish v1" priority:high due_date:2026-02-01
new milestone "Docs complete" goal:"Publish v1" weight:2
  • Milestone criteria: count or checklist (see Dashboard â†’ Goals widget).
  • Habits: completion_dates vs incident_dates for good/bad habits.
