Requirements

  • Windows (for .bat launcher and PowerShell helpers).
  • Git (optional, for cloning and updates).

Launch the Console

  1. Open a terminal in the project root.
  2. Run Console_Launcher.bat to open the interactive console.
  3. Or run commands directly: Console_Launcher.bat <command> [args...]

Your First Day

  1. Set status: status energy:high focus:good
  2. Generate schedule: today reschedule
  3. View schedule: today

Common Workflows

Create a Project

new project "My New Project"
new task "Task 1 for my new project" project:"My New Project"
list task project:"My New Project"

Create a Routine

new routine "My Evening Routine"
new subroutine "Wind Down" routine:"My Evening Routine"
new microroutine "Read a book" subroutine:"Wind Down"
add "My Evening Routine" to "Monday"
