Settings

Chronos settings live under User/Settings. Prefer lowercase filenames; older TitleCase files are still supported.

Core Settings Files

  • points_settings.yml - points ledger configuration (backward compatible with Points.yml).
  • status_settings.yml - status types and weights.
  • theme_settings.yml - console themes.
  • timer_settings.yml and timer_profiles.yml - focus timer profiles.

Defaults

  • <type>_defaults.yml - default values for item creation.
  • Chronos prefers lowercase: task_defaults.yml, goal_defaults.yml.

Editing Settings

  • Use the Dashboard Settings widget for YAML validation.
  • Or edit files directly; changes are picked up immediately.
  • CLI helper: settings <file_shortcut> key value.
