Settings

Chronos settings live in User/Settings/. Prefer lowercase filenames; older TitleCase files are still supported.

Conventions

  • points_settings.yml
  • achievement_defaults.yml
  • <item>_defaults.yml (e.g., task_defaults.yml)
  • Timer: Timer_Settings.yml, profiles in Timer_Profiles.yml

Edit

  • Use the Dashboard Settings widget to edit files with YAML validation.
  • Or edit on disk; the engine will pick up changes immediately.
