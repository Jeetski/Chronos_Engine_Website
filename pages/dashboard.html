Dashboard Guide

How to run and use the Chronos Dashboard: views, widgets, and API endpoints.

Run

  • From the Chronos console, run: dashboard
  • Or double-click: dashboard_launcher.bat

Views & Widgets

  • Calendar: Year/Month/Week/Day; select a block to target edits
  • Today: Trim '5/'10/custom; Change; Cut; Mark; Reschedule
  • Notes, Status, Timer, HabitTracker, GoalTracker, ItemManager, DebugConsole
  • Settings: edit User/Settings/*.yml with validation

Template Builder

See the Template Builder guide for editing week/day/routine/subroutine/microroutine templates with drag-and-drop and strict nesting rules.

API (selected)

  • POST /api/cli " run commands
  • GET /api/today, POST /api/today/reschedule
  • GET /api/items, GET /api/item, POST /api/item*
  • GET /api/goals, GET /api/habits
  • GET /api/timer/status|profiles|settings, POST /api/timer/*
  • GET/POST /api/settings " list/read/write settings files
