Template Builder

Design and edit reusable templates (week, day, routine, subroutine, microroutine) with drag-and-drop.

Open

  1. Run the Dashboard server and open the UI (see Dashboard Guide).
  2. Use the top bar: View ' Template Builder.

Library

  • Types: item (aggregates leaf content like task, note, goal, etc.), microroutine, subroutine, routine, day, week.
  • Subtype filter: appears when item is selected to narrow by leaf type.
  • Clicking adds to the template if allowed; otherwise loads details in the Inspector for review.

Editing

  • Select template type (routine/subroutine/microroutine/day/week) and name, then Load.
  • Drag to reorder; drop into to make a child; drop before/after for same-level; drop near the left edge to outdent.
  • Use the Inspector to edit type, name, duration (number or parallel), ideal start/end, and depends_on.
  • Each node shows a duration badge: for sequential (sum), for parallel (max with children).

Nesting Rules

  • Order of size: week > day > routine > subroutine > microroutine > item (task/note/etc.).
  • You cannot nest a bigger template under a smaller one (e.g., routine under subroutine is blocked).
  • You cannot nest the same kind within itself (routine inside routine, day inside day, week inside week).

Saving

  • Click Save to write children: to the template YAML via the Dashboard API.
  • APIs: GET /api/template/list?type=TYPE, GET /api/template?type=TYPE&name=NAME, POST /api/template with YAML body { type, name, children }.
