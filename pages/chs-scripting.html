CHS Scripting

Run sequences of Chronos commands from .chs files. Highlights:

  • Variables: set var name:World, then use @name or @{name}
  • Escaping: write @@ for a literal @
  • Properties: key:value tokens parsed as command properties
  • Conditions: if ... then with elseif, else, end; supports parentheses
# morning.chs
status energy:high focus:good
today reschedule
if exists task:"Deep Work" then
  append note Prep "Outline goals for the block"
end

See more examples in the repository.
